Style text selection and highlighting

Whether selecting text for copying, searching, or simply focusing on information, the highlight effect provides a clear visual cue.

By default, browsers apply a selection style—typically changing the background and text color—but did you know you can customize this? Maybe you want to improve contrast for readability, match your site’s design, or just add a cool visual effect.

That’s where the ::selection pseudo-element comes in. It allows you to override the browser’s default styling and apply your own colors, effects, or other visual changes when users highlight text.
