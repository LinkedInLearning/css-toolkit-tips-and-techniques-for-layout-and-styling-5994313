Understanding the line-height property

The line-height property sets the height of the line box, which surrounds each line of text. Most browsers default to a value between 1.15 and 1.2, meaning the font size is multiplied by this factor. This default often feels too tight for smaller text, while larger font sizes may benefit from a slightly reduced line height for better readability and appearance.

Line height controls the adjustable height of the line box around text.
